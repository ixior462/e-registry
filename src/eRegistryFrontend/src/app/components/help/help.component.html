Help

This is an E-Registry Application.

If you are a moderator you can view and create user entities as well as view and create classes with multiple users. If you are a teacher you can view your classes and manage grades. If you are a student you can view your grades.

To navigate through service, use controls on panel on the top of site.

You can view and add users by choosing the Users section.

  • If you choose to browse users you can see the list of users with their names, last names, roles (teacher / pupil). You can also edit or delete users in the right hand side action panel.
  • If you choose to add new users you need to provide their name, surname, login and role (teacher / pupil).

You can view and add classes by choosing the Classes section.

  • If you choose to browse classes you can see the list of classes with their names and grades. You can also edit or delete classes in the right hand side action panel.
  • If you choose to add a new class you need to provide its name and grade. Then you can choose which of the currently unassigned users should be moved to the newly created class.
    • In order to move particular students to the new class you need to click on their names so that their "move" value will be set to true. Then you need to click on the arrow pointing right.
    • In order to remove particular students from the new class you need to click on their names in the list on the left side so that their "move" value will be set to true. Then you need to click on the arrow pointing left.

To log out or in order to see your own user info, such as login, you can click on User Info section on the right hand side of the top control panel.
